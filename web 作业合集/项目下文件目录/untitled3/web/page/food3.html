牛肉炒青椒

牛肉炒青椒是一道家常菜，口感鲜嫩多汁，青椒的清脆与牛肉的鲜香相得益彰。

食材

  • 200克牛肉
  • 2个青椒
  • 适量葱姜蒜末
  • 适量盐和生抽
  • 适量淀粉和料酒
  • 适量色拉油

步骤

  1. 将牛肉切成薄片，加入葱姜蒜末、盐、生抽、淀粉和料酒腌制15分钟。
  2. 青椒洗净后去籽，切成条状。
  3. 热锅冷油，将腌制好的牛肉倒入锅中煸炒至变色，盛出备用。
  4. 原锅再次热油，加入青椒条煸炒至变色，加入牛肉继续翻炒均匀。
  5. 根据个人口味，可以适量添加盐和生抽调味。
  6. 炒好的牛肉炒青椒盛出，装盘即可享用。
